บทนำ: การใช้ Anchor Link

Anchor Link เป็นคุณสมบัติที่มีประโยชน์ใน HTML ที่ช่วยให้ผู้ใช้สามารถนำทางไปยังส่วนต่างๆ ภายในหน้าเว็บเดียวกันได้อย่างรวดเร็ว โดยไม่ต้องเลื่อนหน้าจอเองทั้งหมด

หลักการคือการกำหนด ID ให้กับองค์ประกอบเป้าหมาย และสร้างลิงก์ที่ชี้ไปยัง ID นั้นด้วยเครื่องหมาย #

รายละเอียดเพิ่มเติมเกี่ยวกับการทำงาน

เมื่อคุณคลิกลิงก์ที่มีรูปแบบ href="#ชื่อID", Web Browser จะมองหา Element ที่มี ID นั้นๆ และจะทำการเลื่อนหน้าจอให้ส่วนนั้นปรากฏขึ้นมาที่ด้านบนของหน้าต่าง Browser โดยอัตโนมัติ

คุณสามารถใช้ ID กับ Element ใดก็ได้ เช่น Heading, Paragraph, Div หรือแม้แต่รูปภาพ

สรุปและข้อควรจำ

การใช้ Anchor Link ทำให้หน้าเว็บของคุณใช้งานง่ายขึ้น โดยเฉพาะสำหรับหน้าที่มีเนื้อหาเยอะๆ

  • ID ต้องไม่ซ้ำกันในหนึ่งหน้า HTML
  • ลิงก์ต้องขึ้นต้นด้วย # ตามด้วย ID
  • ช่วยปรับปรุงประสบการณ์ผู้ใช้ (UX)
